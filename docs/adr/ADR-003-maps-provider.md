# ADR-003: Провайдер карт

- **Статус:** Accepted
- **Дата:** 2026-07-20
- **Автор:** Adel Gabdullin

## Контекст
Нужна интерактивная карта с маркерами мест на фронтенде. Важны: отсутствие обязательного платного ключа на старте, гибкость и open-source.

## Решение
**Leaflet** с открытыми тайлами (OpenStreetMap) на этапе MVP.

## Рассмотренные альтернативы
- Google Maps — требует биллинг-ключ и имеет лимиты, избыточен для MVP.
- Яндекс Карты — хорошая локализация под РФ, но жёстче по лицензии и интеграции; кандидат на будущее.

## Последствия
Плюсы: бесплатно на старте, простая интеграция с React, полный контроль. Минусы: качество тайлов OSM в отдельных регионах ниже коммерческих; провайдера можно сменить позже (решение изолировано на уровне frontend).
