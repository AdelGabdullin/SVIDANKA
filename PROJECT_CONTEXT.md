# SVIDANKA Project Context

## Название проекта
Свиданка

## Идея
Сервис поиска интересных мест для свиданий, прогулок и совместного отдыха.
Пользователь выбирает город, бюджет, категорию и получает подходящие места
на карте или через режим случайного выбора («Кубик»).

## Проблема
Люди хранят идеи мест в разных источниках (Instagram, Telegram, TikTok, карты,
заметки). Информация теряется, сложно быстро выбрать место.

## MVP
- города
- карта
- места
- категории
- бюджет
- отзывы
- предложения новых мест
- модерация
- случайный выбор места

## Ограничения MVP
- нет регистрации
- нет сбора персональных данных
- отзывы анонимные

## Цель проекта
Полноценный коммерческий продукт: backend, frontend, тестирование, CI/CD,
документация, мониторинг.

## Ключевые решения
См. `docs/adr/`:
- ADR-001 — технологический стек (FastAPI)
- ADR-002 — PostgreSQL
- ADR-003 — Leaflet вместо Google Maps
